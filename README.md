# Sales Analysis Project using Microsoft Excel

## Project Overview

This project focuses on analyzing sales data and evaluating sales representatives' performance using Microsoft Excel. The project demonstrates practical Excel skills including data validation, sales analysis, Pivot Tables, lookup functions, and incentive calculations based on predefined business rules.

## Objectives

* Validate and clean sales data.
* Analyze sales performance for each representative.
* Calculate incentives according to sales targets.
* Identify top and low-performing sales representatives.
* Create dynamic reports using Excel tools.

## Features

* Data validation and duplicate checking.
* Sales summary using Pivot Tables.
* VLOOKUP functions for retrieving representative information.
* Task Form for displaying representative details.
* Maximum and Minimum sales identification.
* IF and AND functions for business rule implementation.
* Automated incentive calculation based on sales thresholds.

## Business Rules

* Sales ≥ 10,000 → 2% incentive.
* Sales ≥ 20,000 → 3% incentive.
* Sales ≥ 40,000 → 4% incentive.
* Sales ≥ 50,000 → Fixed bonus.
